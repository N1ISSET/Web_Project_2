1. To link your database you need to change the password in file: com.pe_project.saveeverychildren.common.DatabaseConnection
2. To create database, open database_code.txt copy SQL code and paste in your MySQL.
3. To operate these code you are required to import file with springboot framework.
4. To run this project, run file com.pe_project.saveeverychildren.PeProjectApplication
5. Type localhost:8080 to go to the home page
6. You can start using the web client!
7. To login into the admin management system, the default admin name is (admin), the password is (passord).
8. Now you can start managing the database details on the web.
